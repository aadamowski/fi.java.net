
--------------------------
JapexXMLDriverLibrary 
--------------------------

Introduction:

 This is a library that contains a number of Japex drivers for measuring
 the performance of the following:

 o Parsing and serializing of XML and FI documents
 o JAXB marshalling and unmarshalling of XML and FI documents

 This library is used by the Japex configuration files in the
 FastInfosetPerformance directory.

Building:

 >> ant jar
