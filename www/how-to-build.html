How to build

There are two routes to building the FastInfoset source code:

  1. Install NetBeans (version 4.0 or greater) and use either Ant (version 1.6.x or greater) or the IDE to build the project (a NetBeans project is available to open in the IDE); or

  2. Use the alternative ant build script, build-without-nb.xml, for the case where NetBeans is not installed.

After building the FastInfoset.jar will be available in the dist directory of FastInfoset.
